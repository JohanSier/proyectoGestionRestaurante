# Savor
Plataforma web para la gestión integral de restaurantes.  
Optimiza la operación diaria centralizando mesas, productos y pedidos en tiempo real.

---

## 🚀 Visión del Producto

Savor permite a restaurantes pequeños y medianos gestionar su operación de manera simple, rápida y eficiente.

Elimina errores manuales, reduce tiempos de servicio y ofrece visibilidad clara del estado del negocio en todo momento.

---
